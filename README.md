# ThePuzzleSolver
It is a fully responsive Flask web app where you will get a lot of puzzles problem with answer. It is fully responsive website, build using python flask, HTML , CSS, Bootstrap, mysql database. Here you you can login as admin and add modify puzzles/posts. It also has a contact page where anyone can send query or suggestions to admin. It will get stored in the database and also the admin will get notified via email.
